Download and installation

There are a number of convenient ways to obtain Ibis.js

GitHub

For the complete Ibis.js distribution including all docs and demos.

https://github.com/kasargeant/Ibis.js/archive/master.zip

Git

For the complete Ibis.js distribution including all docs and demos.

git clone https://github.com/kasargeant/Ibis.js.git

NPM

For the minimal Ibis.js distribution - library only - nothing superfluous.

npm install Ibis.js

Pick whatever's convenient for you - all published code is in-sync.

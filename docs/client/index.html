Ibis.js

High-performance GeoJSON mapping for data-heavy web applications.

Key Features

What can you do with Ibis.js?

Draw GeoJSON data directly in the browser - without tiles - and quickly!


Add multiple map layers - if needed, from multiple datasources.


Use standard geometry types: Polygon, Line, Point.


Use extended geometry types for richer mapping: PointText, PointIcon.


Create your own plug-ins to add unique functionality to your map.


Everything can all be visually customised by the user - for their target `look 'n feel`.


All core map functionality: pan, zoom etc.


Library written using modern (ES6) JavaScript standards with clean support for extensibility.

Download and installation

There are a number of convenient ways to obtain Ibis.js

GitHub

For the complete Ibis.js distribution including all docs and demos.

https://github.com/kasargeant/Ibis.js/archive/master.zip

Git

For the complete Ibis.js distribution including all docs and demos.

git clone https://github.com/kasargeant/Ibis.js.git

NPM

For the minimal Ibis.js distribution - library only - nothing superfluous.

npm install Ibis.js

Pick whatever's convenient for you - all published code is in-sync.
